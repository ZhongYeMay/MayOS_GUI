MayOS_with_GUI 更新日志

版本：MayOS_with_GUI_0.0.8

发布日期：2025年7月21日

  • 音乐播放器（music.py）VLC依赖自动化：自动检测 system/func/vlc-3.0.20/vlc-3.0.20/ 目录，确保 libvlc.dll、libvlccore.dll、plugins 文件夹齐全。首次启动自动下载并解压 VLC 官方 zip 包，无需手动干预。
  • VLC初始化修复：plugin-path 强制指向完整 bin 目录，启动时自动将 VLC bin 目录加入 PATH 环境变量，彻底解决 python-vlc 找不到 DLL 的问题。避免孤立 DLL 路径导致初始化失败。
  • 用户注册/登录流程优化（start.py）：首次启动强制注册，注册后自动进入登录流程。支持用户名下拉选择，注册/登录成功后自动授权启动主程序 main.py。未授权或窗口关闭时强制退出，保证安全性。
  • 自动下载 ffmpeg.exe、ffplay.exe：启动时自动检测 system/func/ 目录下是否存在 ffmpeg.exe、ffplay.exe，若无则自动下载并解压，显示命令行进度条，已下载则跳过。
  • 异常处理和弹窗提示优化：所有依赖检测、初始化、注册/登录流程相关弹窗和错误提示均已优化，用户体验提升。
  • 代码结构优化：依赖检测更完善，异常处理更健壮，整体稳定性提升。

版本：MayOS_with_GUI_0.0.7

发布日期：2025年7月19日

  • 主界面菜单栏和桌面新增“启动计算器”和“启动音乐播放器”按钮，支持一键启动 calc.py 和 music.py。
  • 音乐播放器：专辑封面联网获取部分改为多线程处理，界面更流畅，API更换为网易云音乐，曲库更丰富。
  • 音乐播放器：增加反调试环境检测、基础字符串混淆和关键字符串加密，提升逆向难度。
  • 优化更新日志显示方式，支持本地HTML文件优先展示，防止显示错位并美化内容。
  • 优化 changelog 展示样式，采用 Google Material Design 3（MD3）风格美化。
