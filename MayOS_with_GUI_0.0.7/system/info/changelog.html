MayOS_with_GUI 更新日志

版本：MayOS_with_GUI_0.0.7

发布日期：2025年7月19日

  • 主界面菜单栏和桌面新增“启动计算器”和“启动音乐播放器”按钮，支持一键启动 calc.py 和 music.py。
  • 音乐播放器：专辑封面联网获取部分改为多线程处理，界面更流畅，API更换为网易云音乐，曲库更丰富。
  • 音乐播放器：增加反调试环境检测、基础字符串混淆和关键字符串加密，提升逆向难度。
  • 优化更新日志显示方式，支持本地HTML文件优先展示，防止显示错位并美化内容。
  • 优化 changelog 展示样式，采用 Google Material Design 3（MD3）风格美化。
