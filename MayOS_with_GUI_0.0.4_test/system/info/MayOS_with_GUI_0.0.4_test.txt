MayOS_with_GUI_0.0.4 Test Release
发布日期: 2025年6月2日14点35分
注:本系统基于"MayOS_with_GUI_0.0.3"修改而来，并非"MayOS_with_GUI_0.0.4"正式版

漏洞修复:
1.修复了系统版本信息无法正常读取的问题
2.修复了更新日志无法正常读取的问题

功能改进:
1.将窗口分辨率调整为"1280x740"，确保壁纸信息显示完整
2.将窗口标题更改为系统版本

新功能:

1.实时时钟功能：
在右下角添加了数字时钟
使用update_clock()函数每秒更新一次时间
时钟使用白色文字黑色背景，确保可见性

2.界面优化：
时钟使用place布局精确定位在右下角
时钟文字使用Arial 12号字体
时钟会自动适应窗口大小变化

3.代码结构改进：
将时钟功能封装为独立函数
使用after方法实现定时刷新
保持原有功能完整

4.使用说明：
时钟会自动启动并每秒刷新
时钟位置固定在窗口右下角，不会随其他内容移动
可以通过修改update_clock()函数中的strftime格式来改变时间显示样式

5.新增"版本管理器"类
专门负责系统版本及更新日志的读取
