MayOS_with_GUI_0.0.3 Stable Release
需要特殊说明的是：本次更新改动幅度相较于上一个版本来说较大
发布日期: 2025年6月2日00点59分

新功能:
1. [新增] Bing每日壁纸自动更新功能
   - 系统启动时自动获取并设置Bing当日精选壁纸
   - 新增"更新Bing壁纸"菜单选项，支持手动更新
   - 壁纸本地缓存保存至 files/background/ 目录
   - 壁纸文件名包含日期信息(bing_YYYYMMDD.jpg)

2. [新增] 壁纸更新状态反馈
   - 底部状态栏显示当前壁纸标题信息
   - 壁纸更新失败时显示友好错误提示

3. [优化] 菜单结构调整
   - "操作"菜单新增"更新Bing壁纸"选项
   - 保持原有菜单结构不变

技术变更:
1. 新增依赖库:
   - requests (用于HTTP请求)
   - Pillow (用于图像处理)

2. 代码优化:
   - 增强错误处理机制
   - 改进文件保存逻辑
   - 保持原有功能兼容性

已知问题:
1. 在无网络连接情况下无法获取Bing壁纸
2. 首次运行时需要联网获取壁纸
3. 需要手动创建 files/background/ 目录

升级说明:
1. 需要安装新依赖:
   pip install pillow requests
2. 建议创建目录:
   mkdir -p files/background
3. 原配置文件兼容，无需迁移

注意事项:
1. Bing壁纸版权归微软所有
2. 每日仅自动更新一次壁纸
3. 手动更新不受限制
