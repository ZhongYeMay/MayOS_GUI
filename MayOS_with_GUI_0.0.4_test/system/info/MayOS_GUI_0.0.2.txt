MayOS_with_GUI_0.0.2 DevRelease
1、该版本存在许多未经测试的漏洞，稳定性未知
2、目前启动脚本还未完成，只完成了注册部分
3、目前多语言支持不完善，当前仅支持英语（美国）、中文（简体）、中文（台湾、香港、澳门）
4、目前系统新增应用[GUI]：音乐播放器 2.0
5、菜单栏：
（1）操作菜单：包含退出选项
（2）帮助菜单：包含官方网址（点击会打开浏览器）和更新日志
（3）关于菜单：显示版本信息
6、背景图片：
（1）加载并显示背景图片
（2）如果图片不存在，会显示黑色背景和错误消息
7、欢迎消息：
（1）控制台和GUI都会显示欢迎消息
（2）帮助命令现在同时在控制台和GUI中显示结果
8、状态栏：
（1）底部添加了状态栏，显示当前状态
9、错误处理：
（1）对文件不存在的情况进行了处理

需要特别说明的是：
1、确保有 files/background/default.jpg 背景图片文件
2、确保有 system/info/MayOS_GUI_0.0.1.txt 帮助文件
3、运行程序后可以通过菜单栏访问各种功能
